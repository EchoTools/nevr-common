# Archived Proto Packages

These proto packages are deprecated and excluded from the buf module.
They are kept here for historical reference only.

- `api/v1/` - REST API definitions (replaced by nakama built-in APIs)
- `apigrpc/v1/` - gRPC service definitions (replaced by nakama built-in RPCs)
- `types/v1/` - Lobby session types (unused)

Note: `apigame/v1` and `rtapi/v1` were renamed to `engine/v1` and `gameservice/v1` respectively and remain active packages in the buf module.
